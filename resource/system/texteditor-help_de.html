Kurzwahltasten im Text Editor

Inhaltszuweisungen

CTRL-E Lokales Datum einfügen, lokales Format (e.g. "Dec 18, 2011 ")
CTRL-F Lokale Zeit einfügen, lokales Format (e.g. "5:12:36 PM ")
CTRL-D Lokales Datum einfügen, Universalformat (e.g. "2011-12-18 ")
CTRL-T Lokale Zeit einfügen, Universalformat (e.g. "17:12:36 ")
CTRL-U Universale Zeit einfügen (e.g. "2011-12-18 16:12:36 UT ")

Auswahltasten

CTRL-W Nächstes Wort auswählen
CTRL-L Aktuelle Zeile auswählen
CTRL-P Paragraphen auswählen
CTRL-A Alles auswählen

Editionstasten

CTRL-Z Rückgängig letzte Aktion
CTRL-Y Wiederherstellen letzte Aktion
CTRL-X,
SHIFT-del 		Ausschneiden (Clipboard)
CTRL-C,
CTRL-ins 		Kopieren (Clipboard)
CTRL-V,
SHIFT-ins 		Einfügen (Clipboard)
CTRL-PLUS Schriftsatz vergrößern CTRL-MINUS Schriftsatz verkleinern
