Shortcut Keys in Text Editor

Content Keys

ALT-D Insert local date, local format (e.g. "Dec 18, 2011 ")
ALT-T Insert local time, local format (e.g. "5:12:36 PM ")
CTRL-D Insert local date, universal format (e.g. "2011-12-18 ")
CTRL-T Insert local time, universal format (e.g. "17:12:36 ")
CTRL-U Insert date/time UTC, universal format (e.g. "2011-12-18 22:12:36 UT ")

Selection Keys

CTRL-W Select next word
CTRL-L Select current line
CTRL-P Select paragraph
CTRL-A Select all

Editing Keys

CTRL-Z Undo last action
CTRL-Y Redo last action
CTRL-X,
SHIFT-del 		Cut (Clipboard)
CTRL-C,
CTRL-ins 		Copy (Clipboard)
CTRL-V,
SHIFT-ins 		Paste (Clipboard)
CTRL-PLUS Enlarge text font
CTRL-MINUS Reduce text font
